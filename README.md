# 📊 Exploratory Data Analysis (EDA) & Business Intelligence Project

## 📌 Objective
This project focuses on performing Exploratory Data Analysis (EDA) and SQL-based Business Intelligence on a retail sales dataset to uncover meaningful insights and support data-driven decision-making.

---

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- VS Code

---

## 📂 Dataset
The dataset contains **9800 rows and 18 columns**, including:
- Order details (Order Date, Ship Mode)
- Customer information (Customer Name, Segment, Region)
- Product details (Category, Sub-Category, Product Name)
- Sales data

---

## 🔍 Steps Performed

### 1️⃣ Data Understanding
- Loaded dataset using Pandas  
- Checked structure, data types, and missing values  

### 2️⃣ Exploratory Data Analysis
- Sales Distribution (Histogram)  
- Category-wise Distribution (Bar Chart)  

### 3️⃣ SQL Analysis
- Category-wise Sales  
- Top 5 Products by Sales  
- Region-wise Sales

## 📈 Key Insights
- Technology category generates the highest revenue  
- West region has the highest sales performance  
- South region has the lowest sales  
- Top 5 products contribute significantly to total revenue  
- Sales distribution is highly skewed with presence of outliers
